## Installation

* `git clone git@https://github.com/miniatureweasle/chat.git`
* `cd chat` 
* `npm install`
* `node src/server.js`
* `npm start`
* visit `http://localhost:8080/`

## Testing

* `cd chat`
* `qunit test`
